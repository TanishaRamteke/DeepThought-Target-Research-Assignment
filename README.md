
# DeepThought Business Analytics Assignment
**Candidate:** Tanisha Ramteke

## Project Overview
This repository contains a strategic research pipeline designed to identify "Federer" companies—mid-sized, high-growth, technically differentiated manufacturers—within the Pune industrial corridor.

## Contents
- **[Target Research Dataset](https://github.com/TanishaRamteke/DeepThought-Target-Research-Assignment/blob/main/DeepThought_Target_Company_Research.xlsx):** A scored list of 25 companies verified against E1/E2 eligibility gates.
- **[Methodology Report](METHODOLOGY.md):** Detailed breakdown of sourcing channels (DSIR, IMTEX, etc.) and ICP judgment logic.
- **[Scale-Up Proposal](SCALE_UP_PROPOSAL.md):** A 30-day blueprint to automate the qualification of 1,000+ companies.

## Tech Stack Used
- **Data Analysis:** Python, Pandas, OpenPyXL.
- **Sourcing:** LinkedIn Sales Navigator, DSIR Registry, Apollo.io.
- **Automation Logic:** LLM-based web-scraping and scoring agents.
